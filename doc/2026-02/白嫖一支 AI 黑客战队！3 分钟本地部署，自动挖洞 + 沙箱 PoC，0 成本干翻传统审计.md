## 🔄 5 步审计工作流（全自动）  
1. Orchestrator → 制定审计策略  
  
1. Recon Agent → 识别框架/入口点  
  
1. Analysis Agent → 结合 RAG 挖掘漏洞  
  
1. Verification Agent → 写 PoC → 沙箱执行 → 失败就自我修正  
  
1. 报告生成 → 剔除误报，输出 PDF / Markdown / JSON  
  
## 🏗️ 系统架构（微服务 + 沙箱）  
```
Vue3 Web ─▶ Go-Zero API ─▶ MongoDB                │                ▼            Redis(队列)                │        ┌───────┼───────┐        ▼       ▼       ▼   Worker1 Worker2 WorkerN   （任意扩容）    │    ▼Docker 沙箱 – 自动拉镜像、执行 PoC、回传结果
```
