1. WordPress plugin Woocommerce Custom Product Addons Pro 安全漏洞（CNNVD-202603-4622）  
  
WordPress和WordPress plugin都是WordPress基金会的产品。WordPress是一套使用PHP语言开发的博客平台，该平台具有在基于PHP和MySQL的服务器上架设个人博客网站的功能。WordPress plugin Woocommerce Custom Product Addons Pro是一个应用插件。  
  
WordPress plugin Woocommerce Custom Product Addons Pro 5.4.1版本及之前版本存在安全漏洞，该漏洞源于对用户提交的字段值清理和验证不当，攻击者利用该漏洞可以远程执行代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://wordpress.org/plugins/woo-custom-product-addons-pro  
  
2. Cisco IOS XE Software 安全漏洞（CNNVD-202603-5028）  
  
Cisco IOS XE Software是美国思科（Cisco）公司的一种网络操作系统。  
  
Cisco IOS XE Software存在安全漏洞，该漏洞源于在TLS连接建立期间，内存资源管理不当，攻击者利用该漏洞可以耗尽内存并导致系统拒绝服务。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://sec.cloudapps.cisco.com/security/center/content/CiscoSecurityAdvisory/cisco-sa-iosxe-tls-dos-TVgLDEZL  
  
3. IBM InfoSphere Information Server 安全漏洞（CNNVD-202603-4668）  
  
IBM InfoSphere Information Server是美国国际商业机器（IBM）公司的一套数据整合平台，具有整合各种渠道获取数据信息的功能。  
  
IBM InfoSphere Information Server 11.7.1.6版本及之前版本存在安全漏洞，该漏洞源于允许以明文形式存储用户凭据和其他敏感信息，攻击者利用该漏洞可以读取敏感信息。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://www.ibm.com/support/pages/node/7266489

1. OpenClaw 安全漏洞（CNNVD-202603-5859）  
  
OpenClaw是一个开源的智能人工助理。  
  
OpenClaw 2026.3.11之前版本存在安全漏洞，该漏洞源于对令牌权限限制不当，攻击者利用该漏洞可以制作更高权限的令牌，从而远程执行代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/openclaw/openclaw/releases  
  
2. LoLLMs 授权问题漏洞（CNNVD-202603-5843）  
  
LoLLMs是一个大型语言与多模态系统。  
  
LoLLMs 2.2.0版本及之前版本存在授权问题漏洞，该漏洞源于/api/files/extract-text端点未强制进行身份验证，攻击者利用该漏洞可以导致系统拒绝服务、信息泄露。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/ParisNeo/lollms/releases  
  
3. Gravity Cloud Services Axon Code 安全漏洞（CNNVD-202603-5692）  
  
Gravity Cloud Services Axon Code是一个智能代码编程助手。  
  
Gravity Cloud Services Axon Code存在安全漏洞，该漏洞源于使用不兼容的命令解析器且未能正确处理转义序列，攻击者利用该漏洞可以远程执行代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://www.matterai.so/
