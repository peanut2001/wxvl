#  微软发布紧急补丁修复 Windows 11 中的关键 RRAS 远程代码执行漏洞  
原创 网络安全9527
                    网络安全9527  安全圈的那点事儿   2026-03-15 07:06  
  
微软于 2026 年 3 月 13 日发布了紧急热补丁更新，解决了 Windows 11 版本 24H2 和 25H2 中的严重安全漏洞。  
  
该更新编号为 KB5084597，目标操作系统版本为 26200.7982 和 26100.7982，修复了 Windows 路由和远程访问服务 (RRAS) 管理工具中的三个令人担忧的缺陷，值得注意的是，它无需重启设备即可完成此操作。  
## 修复 RAAS 远程代码执行漏洞  
  
本次热补丁的核心重点是Windows RRAS 组件中的三个漏洞，该组件是一项管理企业和消费者环境中远程连接和 VPN 功能的服务。  
  
本次修复的三个 CVE 漏洞是：  
- **CVE-2026-25172** — RRAS 管理工具中的一个安全漏洞，允许恶意远程服务器中断服务操作或在连接的设备上执行任意代码。  
- **CVE-2026-25173** — 一个相关的 RRAS 漏洞，具有类似的攻击向量，当受害者连接到攻击者控制的服务器时，可导致远程代码执行或拒绝服务攻击。  
- **CVE-2026-26111** — 另一个 RRAS 安全问题，加剧了上述漏洞的风险，在特定条件下可能允许代码执行。  
在所有三个 CVE 中，常见的攻击场景是攻击者设置恶意服务器，并等待运行 RRAS 管理工具的用户或管理员发起连接。  
  
一旦连接成功，攻击者可以破坏工具的功能，更严重的是，还可以直接在受害者的计算机上执行恶意代码。这种攻击在企业环境中尤其危险，因为远程访问管理是常规操作。  
  
与标准的每月安全更新不同，热补丁旨在将关键修复程序应用于内存中正在运行的进程，而不会中断工作流程。  
  
启用热补丁功能的设备会静默接收并安装更新，无需重启即可生效。这种方法显著减少了停机时间，对于管理大量机器的企业部署而言尤为重要。  
  
需要注意的是，此热补丁仅适用于支持热补丁的设备。接收标准 Windows 更新的设备无法获得此特定补丁包。  
  
微软还将最新的服务堆栈更新 (SSU) — KB5083532，版本 26100.8035 — 与热补丁捆绑在一起，以确保更新基础架构本身保持最新状态。  
## 受影响版本  
  
此更新适用于：  
- Windows 11 版本 25H2（操作系统内部版本 26200.7982）  
- Windows 11 版本 24H2（操作系统内部版本 26100.7982）  
- x64 和 Arm64 架构均涵盖在内  
对于支持热补丁的设备，更新将通过 Windows 更新自动下载并安装，无需手动干预。管理员还可以通过 Microsoft 更新目录或服务器更新服务 (WSUS) 访问该更新包（适用于托管环境）。
